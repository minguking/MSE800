# Week 7 Activity 2 - Aquarium Management

This project manages an aquarium in Auckland with different fish types: Goldfish, Shark, Angelfish, Tuna, and Salmon.

## Description

- Uses design patterns to create and manage fish objects.
- Displays the fish category and the number of fish currently available in the aquarium.
- Includes both Factory and Singleton patterns where appropriate.
- Category consists of Edible, Dangerous, Beautiful

## How to run

```bash
python aquarium.py
```
